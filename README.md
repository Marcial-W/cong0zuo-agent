# 从0做Agent

这个仓库用于保存抖音系列视频「从0做Agent」每一期出现过的代码。

系列目标：从 Agent 基础概念开始，逐步做出一个个人知识库助手 Agent。前期代码会偏伪代码和教学示例，重点是帮助理解 Agent 的结构：任务、模型、工具、执行循环、状态和最终输出。

## 目录

| 期数 | 视频主题 | 代码状态 |
|------|----------|----------|
| Agent 01 | 什么是 Agent | 包含最小 Agent 伪代码 |
| Agent 02 | Agent 和 ChatGPT 的区别 | 本期无 Python 代码，仅保留说明 |
| Agent 03 | 一个 Agent 最少需要哪些部分 | 包含最小 Agent 结构伪代码 |
| Agent 04 | 先不接 API，写一个模拟 LLM | 包含可运行模拟 LLM 示例 |
| Agent 05 | 写一个最小知识库 Agent 的伪代码 | 包含可运行个人知识库 Agent 伪代码 |
| Agent 06 | 让 Agent 整理一段本地资料 | 包含可运行本地资料整理示例 |

## 运行方式

进入对应期数目录后运行 Python 文件，例如：

```bash
python episodes/agent-06/agent_06_local_notes.py
```

当前示例默认只依赖 Python 标准库。

注意：Agent 01 和 Agent 03 是视频中的结构伪代码，用了 `llm` / `model` 这类占位对象，主要用于理解结构；Agent 04 起是可直接运行的示例。

## 说明

- 这里的代码优先服务视频教学，不追求一开始就工程化。
- 前几期会有伪代码；从第 4 期开始逐步进入可运行代码。
- 后续会继续补充每期视频出现的代码。
